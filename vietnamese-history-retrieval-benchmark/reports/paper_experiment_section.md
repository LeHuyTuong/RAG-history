# Paper Experiment Section Draft

## Dataset Construction

We constructed a pilot benchmark from 15 local Vietnamese history PDFs. PyMuPDF extracted 9,123 pages, from which 7,425 page-aware chunks were produced. Two volumes, D003 and D007, appear image-only without OCR and are excluded from the silver-label question set until OCR is available.

## Question Set

The pilot contains 40 silver questions generated through LLM-assisted labeling. Questions cover fact/date, person/entity, event, timeline, cause-effect, policy/concept, citation-grounded, and no-answer categories.

## Ground Truth Labeling

The qrels are silver_qrels, not manually verified labels. The file contains 119 qrel rows. Every row has `verified=false`, `label_source=llm_assisted`, and `label_quality=silver`. Manual audit is required before making final claims.

## Baselines

The completed baselines are BM25, multilingual-E5 dense retrieval, and Hybrid BM25 + multilingual-E5. BGE-M3 was attempted but not completed because encoding was too slow on the available hardware.

## Metrics

We report Recall@5, Recall@10, MRR@10, Evidence Hit Rate@5, Citation Accuracy@5, and No-answer Accuracy. Because labels are silver, all scores are preliminary.

## Experimental Procedure

The pipeline extracted PDF pages, chunked page-aware evidence, generated silver candidates, selected 40 silver questions, expanded qrels with BM25 candidate evidence checking, built retrieval indexes, ran baselines, and evaluated retrieval against silver_qrels.

## Error Analysis

Observed errors include missed date/year evidence, missed named entities, semantic mismatch, OCR/extraction noise, chunking ambiguity, wrong-page retrieval, and no-answer failures. Dense retrieval underperformed BM25 on this silver-label set, while hybrid retrieval improved over E5 but did not surpass BM25 in the primary aggregate.

## Threats to Validity

This is a pilot evaluation on a local 15-PDF corpus. The qrels are silver labels generated by an LLM-assisted process, not manually verified labels. OCR is missing for D003 and D007. Automatically generated questions may contain noisy entities or weak evidence. Scores should not be generalized beyond this corpus. A manual audit is needed before using these results in the final paper.
